This setup is my attempt at creating a more uniform template to start from for new projects for me. It allows getting setup fast for a new web project and gives me some tools that make development a breeze. It includes:

  • jQuery

    jQuery is included for DOM manipulation and because it makes most tasks in JavaScript incredibly easier.

  • AngularJS

    AngularJS is included to create a rich data-driven website based on dynamic content and user events.

  • CoffeeScript

    CoffeeScript is included to make writing jQuery and AngularJS more enjoyable and also to make writing them easier and faster.

  • SASS

    SASS is included to make writing CSS less painful and much faster.

  • Compass

    Compass is included to make compiling SASS files super easy.

  • Twitter Bootstrap

    Twitter bootstrap is included for fast scaffolds and moving quickly with best practice HTML.
